****Customer Churn Prediction Project****

**Project Overview:**

Predict which customers are likely to leave a telecom company using machine learning. This project combines data preprocessing, 

modeling, and deployment-ready pipelines to build a robust churn prediction system.

****Project Highlights****

**Dataset:** 

          Telco Customer Churn dataset (customer demographics, services, and account info).

**End-to-End Pipeline:**

          Automated preprocessing → feature engineering → model training → prediction.

**Models:**
          Logistic Regression

**Hyperparameter Tuning:**

          GridSearchCV for optimal model performance.

**Reusable & Deployable:**

          Complete pipeline exported using joblib.

****Tech Stack****

Python (pandas, numpy, scikit-learn)

Jupyter Notebook

Joblib for model persistence

****Outcome****

Identify customers at risk of churning

Enable proactive retention strategies

Demonstrates clean, production-ready ML workflows
